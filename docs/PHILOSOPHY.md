# Why LockWall Is Free

**LockWall is free.** Not "free tier with a paywall later" — genuinely free, with
every feature unlocked, forever. No license keys, no trial, no "Pro" edition.

## Why?

- **Real protection shouldn't be gated.** RDP brute-force is one of the most common
  attacks on internet-facing Windows servers. A tool that stops it belongs in every
  admin's hands, not behind a license.
- **Built for real production use.** LockWall runs on live production servers every
  day. It was built to solve a real problem — sharing it costs nothing and helps
  other admins fighting the same flood of failed logins.
- **Simplicity over monetization.** No company, no billing, no license servers, no
  upsell funnel. Free means the focus stays on making the product *good* instead of
  building paywalls around it.

## What "free" means here

- **Every feature included** — RDP / OWA / SQL protection, web dashboard, RBAC,
  API tokens, GeoIP, Telegram/Email notifications, analytics, audit log. There is no
  "Premium" version holding features hostage.
- **No license keys, no trial, no account, no telemetry.**
- **Free for personal *and* commercial use.**

## But not open source

LockWall is **proprietary freeware**: free to use, but not open source — no modification,
and no public redistribution (re-hosting or mirroring the installer). Please download
only from the official sources (this repo's Releases and lockwall.app). This keeps the
product consistent, trustworthy, and under one roof. See [LICENSE](../LICENSE) for the
full terms.

For trust without open source, every release ships with a published **SHA-256 hash**
and a **VirusTotal scan** — verify before you run.

## Roadmap (all free)

Making the free product better — never locking parts of it away:

- Code signing (no more "Unknown Publisher" warning)
- REST API for automation / SIEM
- Webhooks for external systems
- More notification channels
- The official **lockwall.app** website

Got an idea or found a bug? **Open an issue** — requests and reports are welcome.

## How to support it

It costs nothing. But if LockWall saved your server a headache:

- ⭐ **Star the repo** — visibility helps other admins find it
- 🐛 **Report bugs** and **suggest features** via Issues
- 📣 **Tell another admin** who's fighting RDP brute-force

That's the whole ask. Thank you for using LockWall.
